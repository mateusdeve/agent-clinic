    Create FastAPI router prefix="/api/templates", tags=["templates"].

    Pydantic models:
    - TemplateCreate: nome (str), corpo (str)
    - TemplateUpdate: nome (Optional[str]), corpo (Optional[str])

    Helper function `extract_variables(corpo: str) -> list[str]`:
    ```python
    import re
    ALLOWED_VARS = {"nome", "telefone", "data", "hora", "medico", "especialidade"}
    def extract_variables(corpo: str) -> list[str]:
        found = re.findall(r"\{\{(\w+)\}\}", corpo)
        return [v for v in found if v in ALLOWED_VARS]
    ```

    Helper function `render_template(corpo: str, variaveis: dict) -> str`:
    ```python
    def render_template(corpo: str, variaveis: dict) -> str:
        def replacer(match):
            key = match.group(1)
            if key in ALLOWED_VARS and key in variaveis:
                return variaveis[key]
            return match.group(0)
        return re.sub(r"\{\{(\w+)\}\}", replacer, corpo)
    ```
